# File Organizer

## Overview

This project is a simple Python application that automatically organizes files into separate folders based on their file extensions. It helps keep directories clean and well-structured.

## Features

- Organizes Images
- Organizes Documents
- Organizes Videos
- Organizes Audio files
- Creates folders automatically
- Simple command-line interface

## Technologies Used

- Python 3
- os module
- shutil module

## How to Run

```bash
python file_organizer.py
```
